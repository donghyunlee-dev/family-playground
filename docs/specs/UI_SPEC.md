# UI Specification

## Overview

The platform UI acts as a game hub.

The design should resemble a small app store for games.

---

## Main Pages

Required pages:

/

/login

/lobby

/games

/room/{room_id}

/ranking

/profile

---

## Home Page

Purpose:

Platform entry point.

Components:

- platform title
- login button
- top ranking preview
- game list preview

---

## Lobby Page

Purpose:

Main hub after login.

Components:

- user profile card
- family ranking
- online members
- game catalog

---

## Game Catalog

Each game is displayed as a card.

Card elements:

- game thumbnail
- title
- description
- play button

Example:

Memory Card Game

Find matching cards and score points.

---

## Room Page

Room UI must show:

- game title
- player list
- room status
- host indicator
- start game button

Gameplay area must include:

- main board
- player status
- turn indicator

---

## Game UI Rules

Each game UI must include:

- player list
- current turn
- score display
- game board

---

## Ranking Page

Displays family leaderboard.

Columns:

- rank
- player name
- total score
- games played

---

## Profile Page

User information.

Fields:

- display name
- avatar
- total score
- recent matches
