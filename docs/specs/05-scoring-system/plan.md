# Phase 05 Plan

## Title

Scoring System

## Goal

Persist game results, build ranking views, and expose score history across the platform.

## Implementation Order

1. Finalize result and score persistence schema.
2. Persist game sessions and game scores.
3. Build ranking queries and UI.
4. Add recent match history.
5. Verify scoring consistency across games.

## Dependencies

- playable games from Phase 04
- room completion and result calculation

## Verification

- result persistence tests
- ranking query validation
- score history UI verification

## Exit Criteria

- game results are stored
- leaderboard works
- recent score history is visible
